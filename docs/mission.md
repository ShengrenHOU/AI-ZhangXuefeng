# Mission

## Why This Project Exists

`gaokao-mvp` exists to reduce information asymmetry in the gaokao application process for ordinary Chinese families.

The default users are:
- students and parents in China
- families without strong counseling channels
- families who do not want to pay high fees for志愿服务
- users who need practical, understandable, and revisable guidance

## Product Stance

- AI-first
- workflow as enhancer, not judge
- knowledge-first context, not knowledge-only rigidity
- compassionate but unsentimental
- equal conversation, never condescending
- auditable internally, uncluttered externally

## What This Project Should Help Achieve

- let users talk naturally instead of filling a hidden form
- let the model give direction before every detail is complete
- let formal recommendations improve as dossier and context improve
- let published knowledge and open retrieval strengthen the model rather than replace it
- make trustworthy guidance available without elite assumptions or consulting fees

## What This Project Must Avoid

- guarantee-style admission language
- prestige worship or insider-default assumptions
- turning the model into a mechanical field collector
- exposing engineering traces as user-facing clutter
- letting fragile web retrieval become a hard dependency
